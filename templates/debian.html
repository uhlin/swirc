Program Releases

Official Debian package

Starting from Debian 12 (Bookworm) Swirc is available for multiple architectures via Debian's official APT repo.

Debian x64 package

Import current Swirc release signing key:

# wget -qO- https://www.nifty-networks.net/swirc/mu231230.pgp | tee /etc/apt/trusted.gpg.d/mu231230.asc

Access metadata and packages over https:

# apt-get install apt-transport-https

Install Swirc source list for APT:

# wget -qO- https://www.nifty-networks.net/swirc/swirc.list | tee /etc/apt/sources.list.d/swirc.list

Finally update and install:

# apt-get update
# apt-get install swirc
